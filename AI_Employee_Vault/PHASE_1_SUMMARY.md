## System Architecture

```
Needs_Action/ (Incoming Tasks)
    ↓
Orchestrator.process_needs_action()
    ↓
create_plan_for_task()
    ├─→ ClaudeReasoningEngine
    │   ├─→ Claude API (Opus 4.6) [if ANTHROPIC_API_KEY set]
    │   └─→ Fallback Plan [if no API key]
    ↓
Plans/ (Intelligent Plans)
    ├─ Objective
    ├─ Analysis
    ├─ Steps
    ├─ Approval Requirements
    ├─ Time Estimates
    └─ Risk Assessment
    ↓
Pending_Approval/ (Human Review)
    ↓
Action Executor (Execution)
    ↓
Done/ (Completed Tasks)
    ↓
Logs/ (Audit Trail)
```
