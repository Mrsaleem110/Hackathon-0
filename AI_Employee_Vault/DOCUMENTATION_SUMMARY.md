## 🎯 System Overview

### What is AI Employee Vault?

An **autonomous AI employee system** that:

```
DETECTION (Layer 1)
    ↓ Monitors: Gmail, WhatsApp, LinkedIn, Facebook, Instagram
    ↓
PLANNING (Layer 2)
    ↓ Claude API creates intelligent plans
    ↓
APPROVAL (Layer 3)
    ↓ Human reviews and approves
    ↓
EXECUTION (Layer 4)
    ↓ Sends emails, posts, replies
    ↓
LOGGING (Layer 5)
    ↓ Records everything
    ↓
REPORTING (Layer 6)
    ↓ Weekly audits & CEO briefings
```

## 🔧 Configuration

### Environment Variables (.env)
```bash
ANTHROPIC_API_KEY=your_claude_api_key
GMAIL_CREDENTIALS_PATH=path/to/credentials.json
GEMINI_API_KEY=your_gemini_key (optional)
```

### System Settings (config.py)
```python
# API Keys
ANTHROPIC_API_KEY = "your_key"

# Platforms
ENABLE_GMAIL = True
ENABLE_LINKEDIN = True
ENABLE_WHATSAPP = True
ENABLE_INSTAGRAM = True
ENABLE_FACEBOOK = True

# Audit Settings
AUDIT_SCHEDULE = "0 17 * * 5"  # Friday 5 PM
AUTO_EMAIL_AUDIT = True
AUTO_SAVE_AUDIT = True
```

### Business Rules (Company_Handbook.md)
```markdown
# Email Response Rules
- Always be professional
- Respond within 24 hours

# LinkedIn Posting Rules
- Post 3 times per week
- Focus on industry insights
```

## 🆘 Troubleshooting

### Common Issues

**"No API key found"**
```bash

```

**"Gmail authentication failed"**
```bash
python gmail_login.py
```

**"Dashboard not showing data"**
```bash
python orchestrator.py
# Wait 5 minutes
python advanced_dashboard.py
```

**"Audit generation failed"**
```bash

# Check Logs/ folder for errors
```
