## 🚀 Deploy to Vercel in 3 Steps

### Step 1: Install Vercel CLI
```bash
npm install -g vercel
```

### Step 2: Navigate to web-ui
```bash
cd web-ui
```

### Step 3: Deploy
```bash
vercel
```

That's it! Your UI will be live on Vercel in minutes.
