## 🎯 How It Works

### Tampermonkey Flow
```
Browser loads page
        ↓
Tampermonkey injects button
        ↓
User sees "✨ Create Post" button
        ↓
Click button
        ↓
Script finds native post button
        ↓
Clicks automatically
        ↓
Post editor opens
        ↓
User writes and publishes
```

### Python Integration Flow
```
Run script
        ↓
Choose option
        ↓
Create post template
        ↓
Post saved to Needs_Action/
        ↓
Review and approve
        ↓
Move to Pending_Approval/Approved/
        ↓
Orchestrator publishes
        ↓
Logged to audit trail
```
