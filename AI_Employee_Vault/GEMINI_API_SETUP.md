# Google Gemini API Setup Guide

## Overview

The AI Employee Vault now supports **Google Gemini API** as the primary AI provider, with Claude API as a fallback option. Gemini is preferred and will be used if the API key is configured.

## Quick Setup

### 1. Get Your Gemini API Key

1. Visit: https://makersuite.google.com/app/apikey
2. Click "Create API Key"
3. Copy your API key

### 2. Configure Environment Variable

**Option A: Set Environment Variable (Recommended)**
```bash
export GEMINI_API_KEY=your_gemini_api_key_here
python orchestrator.py
```

**Option B: Add to .env File**
```bash
# Edit .env file
GEMINI_API_KEY=your_gemini_api_key_here

# Run
python orchestrator.py
```

### 3. Verify Setup

```bash
python orchestrator.py
# Look for: "✓ GEMINI API client initialized"
```

## Provider Priority

The system checks for AI providers in this order:

1. **Gemini API** (if GEMINI_API_KEY is set) ← **Preferred**
2. **Claude API** (if ANTHROPIC_API_KEY is set)
3. **Fallback Mode** (basic plan generation)

## Troubleshooting

### Issue: "GEMINI_API_KEY not found"

**Solution**: Set the environment variable
```bash
export GEMINI_API_KEY=your_key_here
python orchestrator.py
```

### Issue: "No AI provider available"

**Solution**: Set either GEMINI_API_KEY or ANTHROPIC_API_KEY
```bash
export GEMINI_API_KEY=your_gemini_key
# OR
export ANTHROPIC_API_KEY=your_claude_key
```

### Issue: API calls failing

**Solution**: Verify API key is valid
1. Check key at https://makersuite.google.com/app/apikey
2. Ensure key has no extra spaces
3. Try with a fresh key

### Issue: Slow responses

**Solution**: Normal behavior
- Gemini API: 2-4 seconds per plan
- Fallback mode: <100ms per plan

## Security Best Practices

1. **Never commit API keys**
   - .env file is in .gitignore
   - Use environment variables in production

2. **Rotate keys regularly**
   - Generate new keys periodically
   - Revoke old keys

3. **Monitor usage**
   - Check API usage dashboard
   - Set up alerts for unusual activity

4. **Use environment variables**
   - Safer than hardcoding
   - Easy to rotate

## Switching Providers

### Switch from Claude to Gemini

```bash
# 1. Get Gemini API key from https://makersuite.google.com/app/apikey
# 2. Set environment variable
export GEMINI_API_KEY=your_gemini_key

# 3. Run orchestrator (will use Gemini)
python orchestrator.py
```

### Switch from Gemini to Claude

```bash
# 1. Unset Gemini key
unset GEMINI_API_KEY

# 2. Set Claude key
export ANTHROPIC_API_KEY=your_claude_key

# 3. Run orchestrator (will use Claude)
python orchestrator.py
```

## FAQ

**Q: Which API should I use?**
A: Gemini API is recommended (free tier available). Claude API is a good fallback.

**Q: Can I use both APIs?**
A: Yes, but only one will be active at a time (Gemini takes priority).

**Q: What if I don't have an API key?**
A: System works in fallback mode with basic plan generation.

**Q: How much does Gemini API cost?**
A: Free tier available with usage limits. Check https://ai.google.dev/pricing

**Q: How do I switch providers?**
A: Just set/unset environment variables and restart.

**Q: Is my data secure?**
A: Yes, API keys are not stored in code. Use environment variables.
